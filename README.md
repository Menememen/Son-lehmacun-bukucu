# 👾 Discord Pokémon Battle Bot

![Discord](https://img.shields.io/badge/Discord-7289DA?style=for-the-badge&logo=discord&logoColor=white)
![Python](https://img.shields.io/badge/Python-3776AB?style=for-the-badge&logo=python&logoColor=white)

Selam! Bu proje, Discord üzerinden kendi Pokémon'unu yakalayıp eğitebileceğin, onlarla antrenman yapıp besleyebileceğin interaktif bir bot projesidir. **Nesne Tabanlı Programlama (OOP)** ve **Çok Biçimlilik (Polymorphism)** prensipleri kullanılarak geliştirilmiştir.

## ✨ Öne Çıkan Özellikler

*   **Rastgele Pokémon Yakalama:** `!go` komutu ile sihirbaz (Wizard) veya dövüşçü (Fighter) sınıfından bir Pokémon sahibi olun.
*   **Gelişmiş Savaş Sistemi:** Vahşi botlara karşı antrenman yapın ve yeteneklerinizi sergileyin.
*   **Besleme Mekaniği:** Pokémon'unuz acıktığında onu besleyerek canını (HP) doldurun.
*   **Çok Biçimlilik (Polymorphism):** 
    *   **Wizard:** Daha hızlı acıkır ama özel kalkan yeteneği vardır.
    *   **Fighter:** Yemek yediğinde daha fazla can kazanır ve saldırı sırasında ekstra güç kullanır.

## 🎮 Komutlar

| Komut | Açıklama |
| :--- | :--- |
| `!go` | Yeni bir Pokémon yakalar. |
| `!info` | Mevcut Pokémon'unun durumunu ve resmini gösterir. |
| `!feed` | Pokémon'unu besleyerek canını doldurur (Zaman sınırlıdır!). |
| `!antrenman` | Vahşi bir bot Pokémon ile savaş başlatır. |
| `!saldır` | Savaş sırasında rakibe hamle yapar. |
| `!heal` | Pokémon'un canını tamamen yeniler. |

## 🛠️ Kurulum

1.  Bu depoyu klonlayın.
2.  `config.py` dosyanıza Discord Bot Token'ınızı ekleyin.
3.  Gerekli kütüphaneleri yükleyin:
    ```bash
    pip install discord.py aiohttp
    ```
4.  Botu çalıştırın:
    ```bash
    python main.py
    ```
